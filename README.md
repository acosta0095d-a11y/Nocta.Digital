# Nocta.Digital

Sitio web de NOCTA — soluciones digitales integrales para empresas que buscan innovar.

Desde software a medida, aplicaciones web y automatización con IA hasta branding, diseño gráfico, producción audiovisual, publicidad, marketing digital y presencia online.

## Stack

- [Next.js](https://nextjs.org/) 15
- React 19
- TypeScript
- Tailwind CSS

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Estructura

- `app/` — rutas y páginas (App Router)
- `components/` — UI, secciones, layout
- `lib/` — contenido, catálogo de servicios, datos
- `public/` — assets estáticos
- `styles/` — estilos globales
